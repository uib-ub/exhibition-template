# Dokumentasjon

## Work

`<Work />` er en IIIF-viser som kan brukes til å vise digitale IIIF-manifester. Den er bygd med [Clover-IIIF](https://github.com/samvera-labs/clover-iiif). Siste versjon av Nextjs og React fungerer ikke med Clover-IIIF, så vi må bruke v2.12.0.

### Grunnleggende bruk

```tsx
import { Work } from '@/components/work'

<Work manifest="ubb-diplom-0034">
  <Work.Description>
    Dette diplomet er datert 7. august 1348 og handler om et drap i Bergen samme vår. 
    Sysselmann Ivar Andresson gir beskjed til kong Magnus VII Eriksson om at Arne Alfinnson 
    drepte Einar Auga, og gir en detaljert beskrivelse av hendelsen.
  </Work.Description>
  <Work.Link href="https://marcus.uib.no/instance/charter/ubb-diplom-0034.html">
    Se mer i Marcus
  </Work.Link>
</Work>
```

<Work manifest="ubb-diplom-0034">
  <Work.Description>

  </Work.Description>


### Direkte IIIF URL

```tsx
<Work manifest="https://api.ub.uib.no/items/ubb-diplom-0034?as=iiif">
  <Work.Description>
    Dette diplomet er datert 7. august 1348 og handler om et drap i Bergen samme vår.
  </Work.Description>
</Work>
```

### Manifest prop

`manifest` prop kan være:

- **String ID**: `"ubb-diplom-0034"` - automatisk konvertert til UB API URL
- **Direkte URL**: `"https://api.ub.uib.no/items/ubb-diplom-0034?as=iiif"`

## UI komponenter

Vi bruker [Shadcn UI](https://ui.shadcn.com) for å bygge våre komponenter. Bruk CSS variabler for å endre farger og størrelser. Større tilpassinger kan gjøres i selve komponenten, men oppdatering vil overskrive komponenten. Bruk git til å hente endringer fra tidligere versjoner av komponentene.
